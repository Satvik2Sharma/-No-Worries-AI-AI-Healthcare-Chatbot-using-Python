## 🚀 How to Run the Project (Basic Version)
# Developed by SATVIK SHARMA

Follow these steps to run the **No-Worries AI Healthcare Chatbot** locally:

### 1️⃣ Open Project Folder
Navigate to the project directory:
```bash
cd healthcare-chatbot
```

### 2️⃣ Activate Virtual Environment
```bash
source .venv/bin/activate
```

### 3️⃣ Install Requirements
Using requirements file:
```bash
pip install -r requirements.txt
```

Or install manually:
```bash
pip install flask pandas scikit-learn openpyxl
```

### 4️⃣ Run the Application
```bash
python app.py
```

### 5️⃣ Open in Browser
Go to:
```text
[http://127.0.0.1:5000](http://127.0.0.1:5000)
```
